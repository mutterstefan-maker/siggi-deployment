# Siggi AI

Ein intelligenter KI-Assistent für E-Mail-Management, Website-Audits und geschäftliche Workflows.

**Live Demo:** https://www.stean.info

## Features

- 📧 **Intelligentes E-Mail-Management** - KI-gestützte E-Mail-Verarbeitung und -Organisation
- 🔍 **Website Audit** - Automatische Webseiten-Analyse
- 📊 **Statistiken & Insights** - Business Intelligence Dashboard
- 🤖 **SIGGI AI Assistant** - Dein persönlicher KI-Assistent
- 💬 **Multi-Channel Support** - Instagram, E-Mail und mehr
- 🔐 **Sicher & Zuverlässig** - Enterprise-Grade Sicherheit

## Tech Stack

- **Frontend:** Next.js 15, React 19, TypeScript, Tailwind CSS
- **Payment:** Stripe Integration
- **Languages:** TypeScript, JavaScript

## Installation & Entwicklung

### Voraussetzungen
- Node.js 18+
- npm oder yarn

### Setup

```bash
npm install
npm run dev
```

Verfügbar unter: http://localhost:3000
